## 3. API Client Configuration (Step 4)

### Before fix
- **File:** `cmd/nexus/tui.go:49`
- **Base URL:** Constructed from `cfg.Daemon.Bind` + `cfg.Daemon.Port` in `~/.nexus/Nexus/daemon.toml`
- **Admin token:** From `cfg.ResolvedAdminKey` in the same config file
- **Problem:** Running daemon uses config at `D:\Test\BubbleFish\v010-dogfood\home\daemon.toml` with a **different admin token**. TUI gets 401 on every authenticated request.

### Root cause
Two config directories:
- TUI reads: `~/.nexus/Nexus/daemon.toml` → token `bfn_admin_REDACTED...`
- Running daemon: `D:\Test\BubbleFish\v010-dogfood\home/daemon.toml` → token `bfn_admin_REDACTED...`

Token mismatch causes silent 401 failures on all `/api/*` calls.

### Fix applied (commit d7f9a11)
Added env var overrides in `cmd/nexus/tui.go`:
- `NEXUS_API_URL` overrides the base URL (default: `http://<bind>:<port>`)
- `NEXUS_ADMIN_TOKEN` overrides the admin bearer token from config
